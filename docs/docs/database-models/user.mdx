# User

The **User** model represents an individual account in the Terrrrr backend. Users can create worlds, join worlds, participate in events, RSVP to events, and author notes.

## Relationships

- **User** has many **WorldMemberships** (can join or create worlds).
- **User** can create many **Events**.
- **User** can RSVP to many **Events** via **EventRSVP**.
- **User** can author many **Notes**.

  "RSVPs": [{ "eventId": "event1" }],
