# Boss

The **Boss** model represents a boss entity in the Terrrrr backend. Bosses are linked to worlds and their progress is tracked via the `WorldBossProgress` model.

## Relationships

- **Boss** has many `WorldBossProgress` records (one per world).
- Linked to worlds through `WorldBossProgress`.

## Example Record

```json
{
  "id": "boss1",
  "name": "Dragon Lord",
  "health": "5000",
  "stage": "Final",
  "worldProgress": [
    {
      "id": "progress1",
      "worldId": "world123",
      "killed": false
    }
  ]
}
```
